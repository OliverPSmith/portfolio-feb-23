A bit about me

Hi, I'm Oliver.

I'm a self-taught front-end developer based in London.

Passionate about creating user-friendly and stylish web projects, I am also a self-taught designer.

For web apps and websites I use Javascript and React.

For styling I use CSS.

And for my designing I use Figma.

I consider myself to be an excellent communicator (written & verbal), which helps me produce exactly what the client wants as well as giving them a clear understanding of what can be achieved.

I am also critical and analytical, spotting errors and fine tuning code and UI/UX for the best aesthetic and practical output on projects.

All websites I make are designed to function on both desktop and mobile screen sizes.

Want to get in touch?

If you would like to discuss working together or collaborating in some way then send me an Email and I'll respond ASAP.

Or if you would like to see some more of my work then feel free to check out my Dribbble and Github pages.
